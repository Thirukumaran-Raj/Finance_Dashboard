# 🚀 Zorvyn Finance Dashboard

A modern, responsive, and role-based **Finance Management Dashboard** built using React.
This application allows users to track income, expenses, and gain smart financial insights with a premium UI.

---

## 🔗 Live Demo
[Click here to view the live project]([https://finance-dashboard-xxxxx.vercel.app](https://finance-dashboard-tau-olive.vercel.app/))


## ✨ Features

### 🔹 Core Features

* 📊 Dashboard with total balance, income, and expenses
* 💸 Add transactions (Admin only)
* 🔍 Search, filter, and sort transactions
* 📈 Insights page with smart financial analysis
* 🧠 Category grouping and monthly comparison
* 👥 Role-based access (Admin / Viewer)

---

### 🔹 Advanced Features

* 💾 Local storage persistence
* 📤 Export transactions (CSV / JSON)
* 🌙 Light & Dark mode toggle
* 🎯 Responsive UI (Mobile + Desktop)
* 🎨 Premium UI with glassmorphism design
* ⚡ Smooth animations and transitions

---

## 🛠️ Tech Stack

* ⚛️ React (Vite)
* 🎨 CSS (Custom styling)
* 📦 Context API (State management)
* 🧩 Lucide Icons
* 💾 LocalStorage (Data persistence)

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/YourUsername/finance-dashboard.git
cd finance-dashboard
```

---

### 2️⃣ Install Dependencies

```bash
npm install
```

---

### 3️⃣ Run the Project

```bash
npm run dev
```

## 🔐 Role System

| Role   | Permissions      |
| ------ | ---------------- |
| Viewer | View data only   |
| Admin  | Add transactions |

👉 Role can be switched using the sidebar dropdown.
